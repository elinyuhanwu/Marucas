## Marucas 

This website is a responsive website using CSS to adapt to diffrent screen sizes.

## Features

```css
@media (max-width: 1100px){
    .dishes .grid li.small, .dishes .grid li.large{
        flex-basis: 50%;
    }
}
@media (max-width: 800px){
    .dishes .grid li.small, .dishes .grid li.large{
        flex-basis: 100%;
    }
}
```
